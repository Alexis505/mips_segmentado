# mips_segmentado
Esta carpeta contiene el diseno de un procesador segmentado que procesa un subconjunto de instrucciones mips.  
  
Instrucciones que procesa esta version  
  Instrucciones tipo R:    
&emsp;              add  
&emsp;              sub  
&emsp;              and  
&emsp;              or  
&emsp;              slt (set on less than)  
  
  Load word (LW)  
  Store word (SW)  
  Branch if equal (BEQ)  
  Add immediate (addi)
